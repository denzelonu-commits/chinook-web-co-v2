# Build & Push Skill
1. Create the component in TypeScript (.tsx)
2. Follow Geist font, tight tracking, gradient text conventions
3. Run `npm run build` to verify no errors
4. **Git Workflow:** - [ -f .git/index.lock ] && rm .git/index.lock
   - git add . && git commit -m "Build update" && git push origin main
5. Take a preview screenshot to verify visual output
